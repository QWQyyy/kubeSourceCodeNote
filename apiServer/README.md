## 概述

APIServer提供了 k8s各类资源对象的CURD/watch、认证授权、准入控制等众多核心功能，在k8s中定位类似于大脑和心脏，它的功能包括：

- 提供了集群管理的REST API接口(包括资源CURD、认证授权、数据校验以及集群状态变更)；
- 是所有模块的数据交互和通信的枢纽，各模块的运作都依赖于APIServer

- 提供丰富多样的集群安全管控机制
- 直连后端存储(Etcd)，是唯一与存储后端直接通信的模块
